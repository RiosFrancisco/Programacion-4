## Pasos a seguir:

### Ir a la carpeta cafeteria
```bash
cd .\cafeteria\
```
### Instalar dependencias
```bash
npm install
```
## En terminales distintas:

### Correr la aplicación en modo desarrollo (Vite)
```bash
npm run dev
```
### Ejecutar tests
```bash
npm run test
```
